# Reel Tracker Utility — Construction

This is the **construction** repo for the Reel Tracker Utility (RTU).

## What it does (v1)
- Requires **Name**, **Company/Garage**, **Build Town**
- Scans reel IDs (camera) or manual entry
- Shows **Last Scanned** and a **Scanned Reels** session list (with remove ✕)
- Exports a formatted **Excel (.xlsx)** and opens the share sheet (download fallback)
